What is SeedMiner?

SeedMiner is a tool that allows you to brute-force a console-unique Key. It utilises your Graphics Card's amazing processing power to try out all possible combinations, with one of them being guaranteed to work.
Using this key (or "seed"), it is possible to then sign a DSiWare Game which has been modified to run an exploit on your Device when launching it.

This all is only possible thanks to a flaw in the key calculation, which makes it (almost) predictable - more information on the nitty-gritty details can be found on zoogie's explanation slide here, but long story short: using your LCFS (Local Friend-Code Seed), it is possible to predict your movable.sed key with an accuracy of about +-4000, which we then can "easily" brute-force.

In addition to getting everything you will need to get your movable.sed, this guide will also recommend you a few tools which you can then use to actually inject your modified Game/Save onto your console.
In most cases, you will be able to use an automated service, but if that fails you, you may need to try one of the other tools available to you.
